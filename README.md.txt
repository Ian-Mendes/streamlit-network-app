# Análise de Redes com Streamlit e Pyvis

Esta aplicação interativa permite analisar redes complexas de agentes e afiliações com base em dados de um dataset do Kaggle.

## Funcionalidades

- Visualização interativa do grafo com Pyvis
- Cálculo de métricas estruturais:
  - Densidade
  - Assortatividade
  - Coeficiente de clustering
  - Componentes conectados
- Histograma da distribuição de grau dos nós
- Análise de centralidade:
  - Degree
  - Eigenvector
  - Closeness
  - Betweenness
- Filtros para visualizar subgrafos
